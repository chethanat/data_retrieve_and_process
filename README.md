# Flas Data Processing Application

## Prerequisites
- Python 3.7 or later
- pip 

## Setup

1. Clone the repository using below url
2. Create Virtual environment and activate it
```
python3 -m venv .venv
source .venv/bin/activate
```

3. Install required dependencies from requirements.txt
```
pip3 install -m reuirements.txt
```

4. Run the Flask application
```
python app.py
```
This application will start running locally at 'http://127.0.0.1:5000/'
